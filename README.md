A simple Python project that analyzes business performance using Excel data.

It loads sales, expenses, and product information, calculates profits, shows monthly summaries, and creates visualizations.

## Features
- Reads data from Excel sheets (`sales`, `expenses`, `products`)
- Calculates cost of goods, gross profit, and net profit
- Displays overall business summary
- Generates monthly performance tables
- Creates charts:
  - Monthly net profit (bar chart)
  - Expense breakdown by type (pie chart)
  - Top-selling products (bar chart)

## Technologies Used
- Python 3
- pandas (data processing)
- matplotlib (visualizations)
- openpyxl (Excel support)
